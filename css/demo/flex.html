flex-grow和flex-shrink 计算方式

文章：详解 flex-grow 与 flex-shrink

flex-grow flex_grow_sum<1

flex-grow 计算公式 width=Math.min( children_width+(parent_width-设置children_width_sum) * flex_grow/Math.max(flex-grow-sum,1))

flex-grow flex_grow_sum>1

设置 100px,

width = Math.min(100+(200-100) * 0.5/Math.max(0.8,1),200)

flex-shrink flex_shrink_sum==1

flex-shrink公式: width = Math.min(width- flex_shrink * [(width_sum - parent_width)*flex_shrink_sum/1 ] * width/sum(width*flex_shrink),width)

所有子元素的尺寸一定会被调整到适应父元素的尺寸,min_width,max_width

flex-shrink flex_shrink_sum > 1

Math.min(600-0.1 * (3000-1903) *(0.1+0.2+0.3+0.5)/1* 600 /(600*0.1+600*0.2+600*0.3+1200*0.5),600)

flex-shrink flex_shrink_sum <1

Math.min(600-0.1 * (2400-1903) *(0.1+0.2+0.3)/1* 600 /(600*0.1+600*0.2+600*0.3+600*0.5),600)
